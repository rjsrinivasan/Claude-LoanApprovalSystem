🚀 HOW TO GET STARTED (5 MINUTES)
================================================================================

1. SETUP DATABASE
   - See QUICKSTART.md Step 1
   - Creates loanapproval_db with all tables

2. CONFIGURE ENVIRONMENT
   - cp .env.example .env
   - Edit .env with your MySQL password and Anthropic API key

3. INSTALL DEPENDENCIES
   - python3 -m venv venv
   - source venv/bin/activate
   - pip install -r requirements.txt

4. RUN SERVICES
   Terminal 1: python -m uvicorn api.main:app --reload
   Terminal 2: streamlit run frontend/streamlit_app.py

5. ACCESS SYSTEM
   - UI: http://localhost:8501
   - API: http://localhost:8000
   - Docs: http://localhost:8000/docs

6. TEST WITH EXAMPLE
   - Use test_payload.json or submit via UI
   - Verify decision is generated
   - Check database for audit trail

Full instructions: See README.md or QUICKSTART.md
